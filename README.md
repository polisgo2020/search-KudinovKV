# search-KudinovKV

# Build

> go build main.go

# Config environment

> set LOG_LEVEL=debug
>
> set LISTEN=localhost:8080

# Usage

_Build index :_

1. > main.exe build file "path/to/dir" "output/filename"
2. > main.exe build database "path/to/dir"

_Search in index:_

1. > main.exe search "index/filename"
2. > main.exe search database
